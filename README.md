# search component

## Установка зависимостей
```
yarn install
```

### Запуск storybook
```
yarn storybook
```

стандартно после запуска вешается на [localhost:6006](http://localhost:6006)

### Запуск приложения
```
yarn serve
```

### Сборка приложения
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
